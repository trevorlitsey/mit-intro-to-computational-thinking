My code for MIT: 6.0002 Introduction to Computational Thinking and Data Science

## Prerequisites

- [Python](https://www.python.org/)
- [pipenv](https://github.com/pypa/pipenv)

### Download

```
git clone
cd mit-intro-comp-sci
pipenv install
```

### Run tests

```
pytest
```

### Run tests in watch mode:

```
ptw
```
